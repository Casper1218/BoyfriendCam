# BoyfriendCam

A mobile app that helps you take better photos with contextual photography tips.

## Overview

BoyfriendCam guides you through selecting a photo scenario (portrait full/half body, close-up) and location (outdoors, indoors, restaurant, beach), then provides relevant photography tips while you take the photo.

Built with Expo and React Native.

## Setup

```bash
npm install
npm start
```

## Development

```bash
npm run android    # Android emulator
npm run ios        # iOS simulator
npm run web        # Web browser
npm run lint       # Run ESLint
```

## Tech Stack

- Expo SDK 54
- React 19 / React Native 0.81
- TypeScript (strict mode)
- expo-router (file-based routing)
- expo-camera & expo-media-library
